### Descrição do objetivo
O projeto consiste no desenvolvimento de uma aplicação web utilizando Python e Flask, com o objetivo
de conectar desenvolvedores e permitir que os usuários cadastrem seus próprios projetos de software,
acompanhem o portfólio de colegas e compartilhem avaliações e feedbacks técnicos por meio de notas e
comentários construtivos de forma simples e organizada.

A proposta é criar um sistema que funcione como um hub colaborativo ou um "portfólio social".
Nele, estudantes ou profissionais de desenvolvimento podem expor suas criações de maneira prática.
Após o cadastro do projeto, informando Nome e Descrição, ele é exibido na página principal do site,
que funciona como uma vitrine ou catálogo.

O sistema busca resolver o problema da falta de canais centralizados para o compartilhamento
de portfólios acadêmicos ou profissionais de forma interativa.
Oferece uma plataforma onde o usuário pode cadastrar, visualizar,
editar e excluir informações de seus próprios projetos de maneira prática, além de permitir
que a comunidade participe ativamente fornecendo retornos em tempo real.

### Justificativa
A criação deste sistema se justifica pela necessidade de organizar e gerenciar o compartilhamento
de portfólios de desenvolvimento, incentivando a troca de feedbacks reais entre estudantes e profissionais
da área de tecnologia. Muitas vezes, projetos de software desenvolvidos de maneira independente ou no ambiente
acadêmico acabam isolados em repositórios pessoais sem qualquer retorno técnico ou avaliação prática de terceiros.

Com essa aplicação, é possível centralizar os dados e os projetos criados em uma vitrine única,
garantindo maior controle, facilidade de acesso, acompanhamento do portfólio de colegas de equipe e
uma excelente
experiência de uso através de uma interface limpa e funcional, baseada em Sidebar, Grid de cartões e
modais dinâmicos.

Além disso, a plataforma conta com um sistema de avaliações seguro e interativo, com nota de 1 a 5 estrelas
e comentários
descritivos, calculando automaticamente a média de notas de cada projeto.
Isso incentiva diretamente o crescimento técnico dos desenvolvedores por meio do feedback de pares e
da revisão colaborativa de forma centralizada.

O projeto possui grande relevância acadêmica, pois permite aplicar de ponta a ponta conceitos fundamentais
de desenvolvimento web, como arquitetura backend estruturada no microframework Flask, mecanismos de rotas,
controle de fluxo e template engine dinâmica com Jinja2, lógica de segurança de sessões com autenticação e
validação de usuários, manipulação de dados e refinamento de design responsivo com CSS Grid e JavaScript.
